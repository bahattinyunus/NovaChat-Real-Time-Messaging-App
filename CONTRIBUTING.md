Thank you for your interest in contributing to NovaChat!

Guidelines:

- Fork the repository and create a feature branch for your work.
- Keep changes focused and small; open a PR to `main` when ready.
- Write clear commit messages and include the motivation for changes in PR descriptions.
- If adding new functionality, include tests when possible.

Reporting issues:

- Create a concise issue with steps to reproduce and expected behavior.

Code style:

- JavaScript: prefer modern ES modules (already used in the repo).
- Keep formatting consistent; run a formatter (Prettier) if available.

If you want help starting a contribution, open an issue and I'll assist.
